# TP DataPress – Projet Exemple

Ce dépôt est un **exemple complet** pour le TP DataPress :
- API minimale (Python / FastAPI)
- Front très simple (HTML statique servi par NGINX)
- Dockerfile multi-stage (API) et images non-root
- docker-compose pour le mode développement
- Manifests Kubernetes (namespace, Deployments, Services, ConfigMap, Secret)
- Pipeline CI minimal (GitHub Actions) pour builder l'image API
- Documentation technique dans `docs/`

> À adapter avec vos propres choix techniques et à compléter avec vos captures, tests, etc.
