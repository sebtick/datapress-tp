# Résumé du contexte DataPress

> À compléter / adapter en fonction des documents fournis dans le sujet.

- DataPress dispose aujourd'hui d'une application monolithique difficile à maintenir.
- La DSI souhaite un POC basé sur Docker et Kubernetes pour :
  - Séparer le front et l'API,
  - Faciliter les déploiements et les tests de nouvelles versions,
  - Préparer une future industrialisation (CI/CD, supervision, etc.).
